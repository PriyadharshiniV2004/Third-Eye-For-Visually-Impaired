# Hardware Connections

## Ultrasonic Sensor
- VCC → IC Pin 8 (Vcc)
- GND → IC Pin 1 (Ground)
- Trigger → IC Pin 3
- Echo → Buzzer circuit

## NE555 Timer IC
- Trigger and Threshold pins shorted
- Reset and Vcc pins shorted
- RC timing circuit using resistors and capacitors
- Output connected to buzzer

## Power Supply
- Battery supply between 3.7V and 5V
- Portable and rechargeable sources recommended
